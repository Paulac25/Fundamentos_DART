# Fundamentos_DART

Este taller tiene como objetivo enseñar los fundamentos de la programación en **Dart**, el lenguaje base de **Flutter**.

Se aprenderá sobre:

- Variables y tipos de datos  
- Estructuras de control  
- Funciones  
- Programación Orientada a Objetos (POO)  
- Manejo de listas y mapas  

También se abordará la estructura y sintaxis del código en Dart.
